Sobre mim

Meu nome é Gabriel, tenho 20 anos, graduando no curso de Análise e Desenvolvimento de Sistemas pelo Centro Universitário de Viçosa - Univiçosa. Sempre gostei de tecnologia, ao entrar neste curso me deparei com a possibilidade de criar sites, aplicativos, plataformas, entre vários outros caminhos. Ao longo dos estudos fui me interessando pelo desenvolvimento web, no momento estou estudando as tecnologias Front-End mais atuais do mercado, porém quero sempre continuar evoluindo e seguir os caminhos do Back-End visando se tornar um Dev Full Stack.

Tecnologias

HTML

Uma linguagem de marcação utilizada na construção de páginas na Web.

CSS

Um mecanismo para adicionar estilos a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags style.

Sass

Uma linguagem de script de pré-processador que é interpretada ou compilada em CSS.

Bootstrap

Um framework web para desenvolvimento de componentes de interface e front-end para sites e aplicações web.

JavaScript

Uma linguagem de programação interpretada estruturada, de script em alto nível.

Typescript

É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem.

Figma

Um editor gráfico de vetor e prototipagem de projetos de design baseado principalmente no navegador web.

PHP

Uma linguagem interpretada livre, muito utilizada para o desenvolvimento de aplicações atuantes no lado do servidor.

MySQL

Um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface.

Python

Uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte.

Git

Um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software.

Github

Uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git
